Bu websitesi nasıl yazıldı

Bu aslında bir websitesi değil. Kendim için obsidian.md üzerinden yazdığım markdown notları.

Şu an içerisinde HTML, CSS, Python ve Javascript bulunan bir projeye dönüştü.

Şunlardan faydalandım:

  1. Obsidian notlarını HTML'ye dönüştürmek için: https://github.com/kmaasrud/oboe
  2. Notlarımın içine yüklendiği bir github repository.
  3. Dosyaların istediğim renkte vesaire olması için bir CSS.
  4. Gereksiz satırları silmek ve linkleri otomatik olarak hyperlinke çevirmek için bir js scripti.

Git kullanımını biraz öğrendim. Şu an yazdığım bat dosyası, özetle şu şekilde:

CHCP 65001
E:
cd zettelhtml
oboe "{notlarimin klasoru}" -t "{templateinadresi}" -o "{ciktiklasoru}" -e
git add .
git commit -a
git push
PAUSE

İlk satırdaki komuta, klasörlerimden birinde "ü" harfi geçtiği için ihtiyaç duydum.

Herkesle paylaşılmaması gereken dosyaları silmeyi yahut .gitignore'a eklemeyi unutmayın.
